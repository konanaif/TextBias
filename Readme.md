- 텍스트 편향성 판단도구의 핵심 기능인 chi-square 테스트를 통한 feature selection과정은 당사의 상용 제품에 임베딩되어 원칙적으로 코드 공개가 불가함.
- 따라서 참고가 될 논문과 유사 코드를 남기는 것으로 공개 과정을 대신하고자 함.
